## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

### Installation

Clone the repository:
```
git clone https://github.com/souvikpramanikgit/LearnHub.git
cd LearnHub
```

Install dependencies:

```
npm install
```

Running the App

```
npm run dev
```

The application will typically be available at http://localhost:3000.
